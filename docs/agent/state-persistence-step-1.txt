Step 1: ConversationStore abstraction

What was added:
- Added AgentConversationStore interface in apps/agent/server/conversation-store.ts
  - getConversation(conversationId)
  - setConversation(conversationId, conversation)
  - getState(conversationId)
  - setState(conversationId, state)
- Added AgentWorkflowState model to persist workflow-level state.
- Added createInMemoryConversationStore() default implementation.

Why:
- Removed direct in-process Map coupling from agent.ts.
- Made memory/persistence backend pluggable and testable.

Validation:
- apps/agent/test/unit/conversation-store.spec.ts
- apps/agent/test/unit/agent-conversation-persistence.spec.ts
